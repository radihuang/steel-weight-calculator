# 鋼構重量計算機

[![GitHub Pages](https://img.shields.io/badge/Demo-GitHub%20Pages-blue)](https://github.com)
[![License](https://img.shields.io/badge/License-MIT-green)]()
[![Language](https://img.shields.io/badge/Language-HTML%2FJS-yellow)]()

一款快速、輕便的線上工具，用於計算鋼構柱與鋼樑的重量。適合結構工程師、建築師、工地主任等專業人士在現場使用。

## 🌟 功能特色

* ✅ **三種鋼材類型支持**

  * TS 方形鋼管（Square Tube）
  * BOX 矩形鋼管（Rectangular Tube）
  * BH H型鋼（H-Beam）
* ✅ **BOX 內隔板計算** ⭐ NEW

  * 支持橫向、縱向、橫縱隔板
  * 自動計算隔板數量（基於間距）
  * 詳細的重量分項顯示
  * 實時更新隔板配置
* ✅ **多種鋼種規格**

  * STK400（普通結構用鋼）
  * STKR400（耐候性結構用鋼）
  * 密度統一採用 7.85 kg/dm³
* ✅ **即時計算**

  * 輸入尺寸後立即顯示結果
  * 單位重量 (kg/m)
  * 總重量 (kg、噸)
  * 重量構成分析（管體 + 隔板）
* ✅ **完全響應式設計**

  * 電腦、平板、手機完美適配
  * 離線使用支持（一次加載後可離線運作）
* ✅ **無需安裝**

  * 純前端應用，直接在瀏覽器運行
  * 無第三方服務依賴

## 📐 計算方法

### TS 方形鋼管 / BOX 矩形鋼管

```
單位重量 (kg/m) = \[(寬 + 高) × 2 - 厚度 × 4] × 厚度 × 7.85 / 1000

例：TS 300×300×9
= \[(300 + 300) × 2 - 9 × 4] × 9 × 7.85 / 1000
= \[1200 - 36] × 9 × 7.85 / 1000
= 82.27 kg/m
```

### BOX 內隔板（新功能）

隔板視為鋼板，根據方向計算面積：

```
隔板面積 (mm²) = 
  - 橫向隔板：BOX寬度 × 隔板厚度
  - 縱向隔板：BOX高度 × 隔板厚度

隔板單位重量 (kg/m) = (隔板面積 × 隔板厚度) × 7.85 / 1,000,000

隔板總重量 = 隔板單位重量 × 長度 × 隔板數量
```

**例：BOX 300×200×9 + 橫向隔板（9mm厚、3片）**

```
管體重量 = 82.9 kg/m
隔板面積 = 300mm × 9mm = 2,700 mm²
隔板單位重量 = (2,700 × 9) × 7.85 / 1,000,000 = 0.191 kg/m
隔板總重量 = 0.191 × 長度 × 3 = 0.573 kg/m

總單位重量 = 82.9 + 0.573 = 83.47 kg/m
```

### BH H型鋼

```
單位重量 (kg/m) = \[2×寬×翼板厚 + (高 - 2×翼板厚)×腹板厚] × 7.85 / 1000

例：BH 600×400×12×22
= \[2×400×22 + (600 - 2×22)×12] × 7.85 / 1000
= \[17600 + 6816] × 7.85 / 1000
= 191.56 kg/m
```

**總重量 = 單位重量 × 長度**

```html

  <option value="stk400">STK400</option>
  <option value="stkr400">STKR400</option>
  <option value="sn400">SN400</option>  <!-- 新增 -->
</select>
```

### 修改密度值

如需使用不同密度（例如不鏽鋼），修改：

```javascript
const density = 7.85; // 改為所需密度值
```

### 新增鋼材類型

在 `updateCalculation()` 函數中新增計算邏輯。

## 📊 精準度說明

* **計算誤差**：±2% （基於標準密度）
* **不包含項目**：

  * 焊接補強板
  * 連接件重量
  * 防火涂料

> 💡 \*\*建議\*\*：重量估算時預留 3-5% 的安全裕度。
